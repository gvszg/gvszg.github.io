專案Demo

Demo 1

購物車網站


My Shop
專案原始碼

DEMO帳號：hello@example.com
DEMO密碼：tkV01pXUt+3T

這個專案要實作一個電子商務網站，運用了User Story規劃出一個個功能來完成。網站使用權限分為管理者與一般使用者。管理者可以新增編輯商品，能上傳圖片至AWS S3，能查看訂單狀態、取消訂單。使用者能瀏覽商品、將商品加入購物車，完成訂購後會收到Email通知。


這個專案用到了：

  • 管理者與使用者權限分離
  • 購物車設計
  • Devise 使用者認證
  • CarrierWave 上傳圖片
  • 歐付寶 API
  • AASM 狀態機
  • Service Object
  • Haml 樣板引擎
Demo 2

TDD練習專案


Yelp Like
專案原始碼

DEMO帳號：hello@example.com
DEMO密碼：tkV01pXUt+3T

這個專案用來練習RSpec測試開發，先寫測試，在寫程式碼。使用者可以註冊、登入，分享店家資訊，並可以選擇給予評價和評語。

測試部分包含：Model層級的單元測試、Controller層級的功能性測試、Capybara模擬使用者觀點的整合測試。


這個專案用到了：

  • TDD/BDD
  • RSpec
  • Capybara
  • 手作使用者認證系統
  • Haml樣板引擎
Demo 3

線上影片網站


My Flix
專案原始碼

DEMO帳號：hello@example.com
DEMO密碼：tkV01pXUt+3T

這個專案實作出一個線上影片網站，運用RSpec作自動化測試執行開發。 網站權限可分為管理者與使用者。管理者能新增影片、圖片與敘述。使用者需先刷卡付費才能完成註冊並收到Email確認通知。登入後可將影片加入等待觀看的清單，可以任意對影片做排序。

測試部分包含：Model層級的單元測試、Controller層級的功能性測試、Capybara模擬使用者觀點的整合測試。


這個專案用到了：

  • TDD/BDD
  • RSpec
  • Capybara
  • 手作使用者認證系統
  • 追隨其他使用者
  • 邀請朋友成為用戶
  • CarrierWave 上傳圖片
  • Stripe API
  • Object Service
  • View decorator
  • Haml樣板引擎
